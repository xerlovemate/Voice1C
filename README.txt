Краткий мануал по использованию программы:

1. После выбора режима необходимо сказать "Старт" и после уведомления начать диктовать
2. Чтобы закончить диктовку скажите "Стоп" или выйдите в главное меню

================================================================================

Программа работает без интернета!!!

В режиме 1С:
Форматирует вашу речь в "ЧтоТоПодобное" 
После знака ; идет перенос строки!
После , идет пробел
До и после = пробелы

Список ГОЛОСОВЫХ команд, которые пригодятся:
Пробел - пробел;
Энтер, интер - enter(перенос строки);
Таб - tab;
Удали - удаляет последнее слово или несколько слов до последнего пробела или знака;
Влево, лево - перемещает курсор на один символ влево;
Вправо, право - перемещает курсор на один символ вправо;
Копье - ctrl + c;
Паста - ctrl + v;
Вырезать - ctrl + x;
Поиск - ctrl + f.


Список поддерживаемых символов:
Восклицательный знак - !
Собака - @
Двойная кавычка - ""
Кавычка - ''
Решетка - #
Доллар - $
Точка с запятой - ;
Процент - %
Двоеточие - :
Амперсант - &
Знак вопроса - ?
Звездочка - *
Скобка - ()
Квадратная скобка - []
Фигурная скобка - {}
Минус, тире - -
Нижнее подчёркивание - _
Плюс - +
Равно - =
Слэш - /
Точка - .
Запятая - ,
Больше - >
Меньше - <
Цифры 1, 2, 3, 4, 5, 6, 7, 8, 9, 0


!!!!!ВАЖНО!!!!! 
В программе не предусмотрены числа. Есть только цифры 1234567890. Например для записи числа
25 следует сказать "два пять", а не "двадцать пять".
